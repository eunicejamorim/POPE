      Point Value

      Determine the point value to be used to calculate the P/L (profit and loss). Futures’ point values depend on whether you are trading directly in a futures exchange or via CFDs. For example, if you are trading S&P 500 futures directly in the CME exchange then for each contract, a change of 1 point equals $50. However, if you are trading S&P 500 futures via CFD, then for each change of 1 point in the S&P futures, assuming the broker uses $1 per point, the point value is 1.
